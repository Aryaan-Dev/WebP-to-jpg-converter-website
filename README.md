# WebP to JPG Converter ⚙️

Welcome to the WebP to JPG Converter website ! This is a lightweight web application that allows you to easily convert #WebP images to #JPG format with adjustable quality settings. Perfect for beginners and pros alike !

## Features ✨

- Drag and drop or browse to upload WebP files.
- Adjust JPG quality with a sleek slider.
- Preview and download converted images.
- Download all converted files as a ZIP archive.
- Responsive design for all devices.

## Folder Structure 📂

To ensure the project runs smoothly, structure your folder like this:

```
WebP-converter/
├── index.html
├── config.js
└── script.js
└── styles.css
```

## Getting Started 🛠️

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, etc.).
- Basic internet connection to load external libraries.

### Installation Steps

1. **Clone the Repository**\
   Open your terminal and run:

   ```bash
   git clone https://github.com/Aryaan-Dev/WebP-converter.git
   ```

2. **Navigate to the Folder**\
   Move into the project directory:

   ```bash
   cd WebP-converter
   ```

3. **Open the Project**\
   Double-click `index.html` to open it in your web browser, or use a live server extension (e.g., Live Server in VS Code) for a smoother experience.

4. **Explore the Files**

   - `index.html`: The main HTML structure.
   - `script.js`: Handles the conversion logic.
   - `styles.css`: Adds the stylish design.

## Usage 🎯

- **Upload Files**: Drag WebP files onto the drop zone or click "Browse Files" to select them.
- **Adjust Quality**: Use the slider to set your preferred JPG quality (0-100%).
- **Convert & Download**: Preview the converted images and download individually or as a ZIP.

## Troubleshooting 🛑

- **Files Not Uploading?**\
  Ensure you're uploading only WebP files. Check the browser console (F12) for errors.

- **Conversion Fails?**\
  Verify your internet connection or try a smaller file size. Large files might need optimization.

- **No Preview?**\
  Clear your browser cache or restart the browser. Ensure `script.js` is loading correctly.

- **ZIP Download Not Working?**\
  Check if the `JSZip` library is accessible (requires internet). Test with a single file download first.
